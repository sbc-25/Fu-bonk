# ⚽ BONK VS Plattform – Das ultimative Fußball-Fan-Erlebnis  

Eine umfassende **Fußball-Fan-Plattform**, entwickelt ausschließlich mit **HTML, CSS und JavaScript**.  
Features: **Live-Videos, NFT-Marktplatz, Amateurfußball-Ligen, soziale Funktionen** und **BONK-Token-Integration**.  

---

## 🛠️ Technische Details  

- ⚡ **Reines HTML/CSS/JS** – kein Framework nötig

## 🚀 Deployment  

### 🌐 GitHub Pages  
1. 📂 Neues Repository erstellen  
2. ⬆️ `index.html` + `README.md` hochladen  
3. ⚙️ Settings → Pages  
4. „Deploy from a branch“ auswählen  
5. Branch **main** + Ordner **/ (root)**  
6. ✅ Live unter: `https://username.github.io/repo-name`  

### 💻 Lokale Entwicklung  
1. `git clone`  
2. `index.html` im Browser öffnen  
3. 🛠️ Kein Build-Prozess notwendig
